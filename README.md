# 🎬 Netflix Customer Churn Analysis

## 📌 Project Overview

This project analyzes customer churn behavior for a subscription-based streaming platform using Python, SQL Server, Machine Learning, and Power BI.

The objective was to identify factors influencing customer churn, uncover business insights, and develop an interactive dashboard for stakeholder decision-making.

---

## 🎯 Business Problem

Customer churn directly impacts revenue and customer retention. Understanding why customers leave can help businesses improve customer experience and increase retention rates.

This project aims to:

* Analyze churn patterns across customer segments
* Identify behavioral trends associated with churn
* Build predictive machine learning models
* Create an interactive dashboard for business users

## 📊 Key Findings

* Customer Churn Rate: **53.9%**
* Laptop users recorded the highest churn count.
* Standard subscription plans showed the highest churn volume.
* Churned customers generated more support queries on average.
* Machine learning models achieved approximately **51% accuracy**, indicating limited predictive power in the available dataset.

---

## 🔍 Machine Learning Insights

Feature Importance Analysis identified:

1. Monthly Income
2. Daily Watch Time
3. Age
4. Support Queries Logged
5. Engagement Rate

as the most influential variables in churn prediction.
